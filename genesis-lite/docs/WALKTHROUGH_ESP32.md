# [BRAIN] Axicor Lite: Пошаговое руководство по запуску на ESP32

Это руководство описывает процесс развертывания нейроморфного движка Axicor Lite на микроконтроллере (LilyGO T-Display или аналоги на базе ESP32/S3). 

**Цель:** Запустить симуляцию 832 нейронов с 26,000+ синапсов в реальном времени (HFT) с использованием Data-Oriented Design и Zero-Copy маппинга.

---

## Этап 1. Подключение и Тулчейн

1.  **Железо:** Вставь USB-кабель в контроллер и подключи к ПК.
2.  **Окружение:** Активируй кросс-компилятор ESP-IDF. Без этого терминал не поймет команду `idf.py`.
    ```bash
    . $HOME/esp/esp-idf/export.sh
    ```
3.  **Порт:** Убедись, что плата видна в системе. Обычно это `/dev/ttyACM0` или `/dev/ttyUSB0`. 
    *Если порт отличается, замени его во всех командах ниже.*

---

## Этап 2. Генерация и Дистилляция Мозга (WTA-Architecture)

Нам нужно создать граф нейронной сети и сжать его до 32 дендритов на нейрон, чтобы он вписался в жесткие лимиты SRAM.

1.  **Сборка ДНК (Baker):** Запусти десктопный компилятор графа. Он создаст "тяжелую" версию модели:
    ```bash
    source .venv/bin/activate
    python3 examples/cartpole_exp/build_brain.py
    ```
2.  **WTA-Дистилляция:** Запусти линковщик памяти. Он отберет ТОП-32 самых сильных синапса для каждого нейрона, внедрит C-ABI заголовок и выровняет бинарник по границе 64 КБ для аппаратного MMU:
    ```bash
    python3 genesis-lite/tools/distill_esp32.py \
      Axicor-Models/CartPole-example/baked/SensoryCortex/shard.state \
      Axicor-Models/CartPole-example/baked/SensoryCortex/shard.axons
    ```
    *Результат:* В корне проекта появятся файлы `shard.sram` (динамические веса) и `shard.flash` (топология).

---

## Этап 3. Сборка и Прошивка Ядра (Dual-Core RTOS)

Теперь мы заливаем саму прошивку и размечаем Flash-память (создаем RAW-раздел `brain_topo`).

1.  **Перейди в папку проекта:**
    ```bash
    cd genesis-lite
    ```
2.  **Сброс конфига (Важно):** Если ты менял настройки Watchdog или частоту RTOS, удали старый кэш:
    ```bash
    rm -f sdkconfig
    rm -rf build/
    ```
3.  **Сборка и прошивка:** Установи таргет (esp32 или esp32s3) и залей ядро:
    ```bash
    idf.py set-target esp32  # или esp32s3 в зависимости от платы
    idf.py -p /dev/ttyACM0 build flash
    ```
    *Примечание:* После этого шага плата уйдет в Boot Loop (Panic), так как раздел с топологией еще пуст. Это нормально.

---

## Этап 4. Zero-Cost Mmap Injection

Мы вшиваем Read-Only топологию прямо во Flash-память. Код будет читать её аппаратно через шину данных, не занимая SRAM.

1.  **Вернись в корень:**
    ```bash
    cd ..
    ```
2.  **Запись топологии:** Запиши `shard.flash` в раздел `brain_topo`:
    ```bash
    parttool.py --port /dev/ttyACM0 write_partition --partition-name brain_topo --input shard.flash
    ```

---

## Этап 5. Запуск Реактора

Открой аппаратный монитор для наблюдения за нейронной активностью:
```bash
cd genesis-lite
idf.py -p /dev/ttyACM0 monitor
```

### Что ты должен увидеть:
*   **Консоль:** MMU детектит заголовок "TOPO", автоопределяет 832 нейрона.
*   **Core 1:** Начинает выдавать логи: `[TICK] Tick X00 | Hot loop time: ~12000 us`. (Если время < 15мс - HFT бюджет соблюден).
*   **Дисплей:** (Позже добавим) Отрисуется зеленый Lock-Free TUI с метриками:
    *   **TPS:** Количество тиков в секунду.
    *   **D:** Текущий уровень дофамина в системе.
    *   **L/R:** Телеметрия моторных команд.

---

## Технический стек (DOD Principles)
*   **SRAM (~120 KB):** Динамические веса, вольтаж, головы сигналов.
*   **Flash (Mmapped):** Статичная топология графа (Zero-Copy).
*   **RTOS Bypass:** Core 1 выделен под физику, Watchdog для IDLE1 отключен.
*   **Branchless OR:** Hit-detection 8 голов аксона без условий (SIMD-style).
*   **1000Hz Tick:** FreeRTOS переведена на миллисекундные кванты для точности HFT.
