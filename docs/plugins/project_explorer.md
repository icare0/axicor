# Project Explorer

**Домен:** `PluginDomain::ProjectExplorer`
**Цель:** Навигация по воркспейсу моделей и генерация междоменных намерений (Intents).

## 1. Междоменная Шина (Event Bus)
Project Explorer не загружает файлы напрямую. Он работает как эмиттер событий для других плагинов:
* Клик по скомпилированному шарду (`.pos`, `.state`) генерирует `ZoneSelectedEvent`, заставляя `ConnectomeViewer` прочитать геометрию.
* Клик по `brain.toml` генерирует `LoadGraphEvent`, пробуждая парсер `NodeEditor`.

## 2. FS Cache
Отрисовка списка файлов опирается на предварительно собранный ресурс `ProjectFsCache`, чтобы не насиловать системные вызовы ОС (`std::fs::read_dir`) каждый кадр рендера UI.
