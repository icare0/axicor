## Описание

Задача управления MuJoCo Ant-v4 (4-ногий робот, 27-мерное пространство действий). Планируемая архитектура:

- **ProprioceptionCortex** — сенсорный ввод (joint angles, velocities)
- **MotorCortex** — 8 моторных команд (torque per joint)
- **RewardCortex** — дофаминовый сигнал на основе forward velocity
