## 2024-05-01 - Optimizing NumPy Packbits
**Learning:** In hot loops where boolean masks need to be packed into bits via multiplication by powers of 2 (e.g., `np.multiply(mask, powers)` then `np.sum(..., axis)`), using a single `np.dot(mask, powers, out=packed)` avoids creating and iterating through intermediate temporary arrays. In `axicor-client`'s zero-allocation design, this gives a ~20-30% speedup while fully respecting the zero-GC requirement (by writing directly to the `out` buffer). While `np.dot` delegates to BLAS for floats, for `uint8` it utilizes NumPy's optimized internal C loop, which is still much faster than a two-step `multiply` + `sum` across memory.
**Action:** When performing `multiply` then `sum` operations to collapse an axis, always consider if `np.dot` (or `np.matmul`/`@`) can be used to fuse the operations and eliminate temporary buffers.
